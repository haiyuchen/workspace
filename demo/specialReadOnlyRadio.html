开发说明

应用1：多个选项值对应一个选项的情况，仅需要选项标签
多个选项，使用a或span标签，name属性值添加同一个合同基本信息中的对应字段，标签内里面填入需要的非选中样式，标签属性添加data-radio属性，值为选项对应的选项值。
多个选项值对应一个选项，该选项的data-radio值以“选项值1,选项值2,选项值3”的形式填写。
组件的首个a或span标签添加target-type="specialReadOnlyRadio"标识，添加fill-style属性，值为选定的样式值，如下：
tick: { 非选中样式: ' ', 选中样式: '✔' }
diamond: { 非选中样式: '□', 选中样式: '■' }
应用2：多个选项值对应一个选项，且需要“其他”选项涵盖其他值，并在“其他”选中时，获取选项值对应的文本
同理应用1，“其他”选项的data-radio值为“other”
“其他”的显示文本使用input标签，name属性值添加合同基本信息中的对应字段，添加data-replace属性，值为使用的约定数据，约定数据请参考“合同数据.xlsx”使用。
应用3：多个选项值对应一个选项，且需要可编辑的“其他”选项
同理应用1，“其他”选项的data-radio值为“other”，而编辑的文本使用basicInput组件
