复选

合同基本信息示例

当前的复选功能不存在约定数据

效果

应用1：
承租人承担费用【 】水费【 】电费【 】网费【 】管理费
应用2：
承租人承担费用【 】水费【 】电费【 】网费【 】管理费【 】

开发代码

应用1： <div>承租人承担费用【<a name="cgzrcdzrChk" data-checkbox="01" target-type="checkbox" fill-style="tick">&emsp;</a>】水费【<a name="cgzrcdzrChk" data-checkbox="02"> </a>】电费【<a name="cgzrcdzrChk" data-checkbox="03"> </a>】网费【<a name="cgzrcdzrChk" data-checkbox="04"> </a>】管理费</div> 应用2： <div>承租人承担费用【<a name="cgzrcdzrChk" data-checkbox="01" target-type="checkbox" fill-style="tick">&emsp;</a>】水费【<a name="cgzrcdzrChk" data-checkbox="02"> </a>】电费【<a name="cgzrcdzrChk" data-checkbox="03"> </a>】网费【<a name="cgzrcdzrChk" data-checkbox="04"> </a>】管理费【<a name="cgzrcdzrChk" data-checkbox="99"> </a>】<input name="basicInput1"></div>

开发说明

应用1：仅需要选项标签
多个选项，使用a或span标签，name属性值添加同一个合同基本信息中的对应字段，标签内里面填入需要的非选中样式，标签属性添加data-checkbox属性，值为选项对应的选项值。
组件的首个a或span标签添加target-type="checkbox"标识，添加fill-style属性，值为选定的样式值，如下：
tick: { 非选中样式: ' ', 选中样式: '✔' }
diamond: { 非选中样式: '□', 选中样式: '■' }
应用2：存在“其他”选项，并且“其他”选项的文本可编辑
同理应用1，“其他”选项的data-checkbox值为配置的对应值，编辑的文本使用basicInput组件


扩展信息示例

应用1：
可编辑复选：【 】A【 】B【 】C【 】D
应用2：
可编辑复选：□A□B□C□D□

开发代码

应用1： <div>可编辑复选：【<a name="hlscsChk1" data-checkbox="01" target-type="checkbox" fill-style="tick">&emsp;</a>】A【<a name="hlscsChk1" data-checkbox="02"> </a>】B【<a name="hlscsChk1" data-checkbox="03"> </a>】C【<a name="hlscsChk1" data-checkbox="04"> </a>】D</div> 应用2： <div>可编辑复选：<a name="hlscsChk2" data-checkbox="01" target-type="checkbox" fill-style="diamond">□</a>A<a name="hlscsChk2" data-checkbox="02">□</a>B<a name="hlscsChk2" data-checkbox="03">□</a>C<a name="hlscsChk2" data-checkbox="04">□</a>D<a name="hlscsChk2" data-checkbox="other">□</a><input name="hlscsChk2" checkbox-input="otherValue"></div>

开发说明

应用1：仅需要选项标签
多个选项，使用a或span标签，name属性值添加同一个不包含在合同基本信息中的随意字段，标签内里面填入需要的非选中样式，标签属性添加data-checkbox属性，值为选项对应的选项值。
组件的首个a或span标签添加target-type="checkbox"标识，添加fill-style属性，值为选定的样式值，如下：
tick: { 非选中样式: ' ', 选中样式: '✔' }
diamond: { 非选中样式: '□', 选中样式: '■' }
应用2：需要有“其他”选项，以及“其他”对应的填写文本框
同理应用1
“其他”选项的data-checkbox值为“other”
“其他”的编辑文本使用input标签，name为该组件的同一个字段，且需添加checkbox-input="otherValue"属性
