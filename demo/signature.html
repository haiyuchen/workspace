签名

扩展信息示例

效果

开发代码

<span name="signature1" target-type="signature" class="signature-show"><img /></span>

开发说明

使用div(换行标签)或span(不换行标签)标签，name属性值添加不包含在合同基本信息中的随意字段，添加target-type="signature"属性标识当前为签名组件，添加class="signature-show"属性为签名事件提供目标。
注意：
签名组件没有固定样式，需要使用者进行开发。例：
.signature-show { width: 200px; height: 150px; border: 1px #d7d7d7 solid; margin-bottom: 10px; }
